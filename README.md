# ddosaptgui
A simple package manager, so you can use apt (or pacman) easily
## Installation
  ### Needed:
      python (at least 3.5)
      colorama for python (pip install colorama)
  ### Main proccess:
      Install requirements
      Open the directory with aptgui.py
      Type in: python (or python3) main.py
# You are done!
